## 🚀 Instalación y Uso

### Prerequisitos
- Node.js 18+ 
- npm o yarn

### Instalación
```bash
# Clonar el repositorio
git clone https://github.com/tu-usuario/mariachis-bogota-elite.git

# Entrar al directorio
cd mariachis-bogota-elite

# Instalar dependencias
npm install

# Iniciar el servidor de desarrollo
npm run dev
```

### Despliegue en Producción
```bash
# Construir para producción
npm run build

# Iniciar el servidor de producción
npm start
```
